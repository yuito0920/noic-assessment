# NOIC 脳アセスメント（Web）

- **サイト**: https://yuito0920.github.io/noic-assessment/

## Firebase（共有データ）

プロジェクト `noic-assessment` で Firestore を有効にし、`index.html` 内の `FB_CONFIG` に SDK 設定を貼り付けてください。

- Firestore コンソール: https://console.firebase.google.com/project/noic-assessment/firestore

未設定時はブラウザのローカル保存のみで動作します。
